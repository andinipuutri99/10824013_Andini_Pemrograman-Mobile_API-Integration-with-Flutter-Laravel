Flutter Laravel API Integration

📌 About Project

Project implementasi integrasi aplikasi mobile menggunakan Flutter dengan backend berbasis Laravel 
melalui REST API. Project ini bertujuan untuk mempelajari komunikasi antara frontend mobile dan 
backend server menggunakan metode HTTP seperti GET, POST, PUT, dan DELETE.

Pada project ini, proses pengujian API dilakukan menggunakan Postman, sedangkan dokumentasi endpoint 
API dibuat menggunakan Swagger agar struktur layanan backend lebih mudah dipahami dan dikembangkan.

Fitur utama yang diimplementasikan meliputi:
- Integrasi Flutter dengan laravel REST API
- CRUD Data
- Authentication sederhana
- Pengujian endpoint menggunakan Postman
- Dokumentasi API menggunakan Swagger
- Komunikasi data berbasis JSON
